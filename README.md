### 🚀 What I'm Building

I'm an MCA graduate from **College of Engineering Trivandrum (CET)** obsessed with one core engineering question:  
**"How can backend systems make financial decisions faster, safer, and smarter?"**

I build production-style services—not toy apps. I focus on real-world patterns like event-driven architectures, distributed caching, and microservices that handle financial data with absolute precision. Finance is a domain where a bug isn't just a bug; it's lost money or broken trust. That constraint drives my passion for strict consistency, audit trails, and idempotency.
